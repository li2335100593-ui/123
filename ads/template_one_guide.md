# 插頁廣告模板 01｜交付說明（繁體中文 / 台灣）

此模板可直接交付給設計或客戶，採「全屏插頁」版型，所有文案與素材皆可快速替換。

## 可替換區塊
- 主標題 / 副標題 / CTA
- 核心賣點與產品文案
- 品牌落款與價格
- 產品圖、背景圖、QR Code

## 字體與語系
- 預設使用「Noto Sans TC」（Google Fonts）。
- 同時提供台灣常用字體備援：PingFang TC、Microsoft JhengHei。

## 快速替換方式（建議）
1. 開啟 `template_one.html`。
2. 搜尋想替換的文字（如「為肌膚定制」），直接替換。
3. 如果要換圖片：
   - 將素材放在同資料夾。
   - 在 `template_one.css` 中替換 `.product` 背景或插入 `<img>`。
4. 保留 `.insert-ad` 的版面，避免破壞排版比例。

## 建議輸出規格
- 版面比例：全屏插頁（自適應各尺寸手機）
- 建議輸出：1080×1920、1242×2688、1290×2796
- 建議出血：3 mm
- 主標最少 24 pt 以上確保可讀性
